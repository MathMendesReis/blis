# Listagem de habilidades do usuário

## Endpoint

`GET /api/users/abilities`

## Cabeçalhos da Requisição

O cabeçalho da requisição deve conter:

- **Authorization** (string): Token de acesso

Exemplo:

```json
{
  "Authorization": "Bearer jwt_token"
}
